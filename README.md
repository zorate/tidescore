# TideScore MVP

Alternative credit scoring platform for Nigerians.

## Features
- Credit score calculation
- Document verification
- Admin dashboard
- Secure authentication

## Technology Stack
- Python Flask
- SQLite database
- Supabase storage
- HTML/CSS/JavaScript

## Setup
1. Clone this repository
2. Install requirements: `pip install -r requirements.txt`
3. Run: `python app.py`

## For Nigerians
This platform is built specifically for the Nigerian financial market.
